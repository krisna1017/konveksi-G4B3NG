Konveksi GABENG Management System
Deskripsi Proyek

Konveksi GABENG Management System adalah aplikasi berbasis web yang digunakan untuk membantu pengelolaan usaha konveksi. Sistem ini memudahkan admin dalam mengelola data pelanggan, pesanan produk, proses produksi, serta pembayaran pesanan.

Aplikasi ini dirancang untuk meningkatkan efisiensi pengelolaan operasional konveksi dan membantu pemilik usaha dalam memantau laporan penjualan serta produksi.

Fitur Sistem
1. Login Admin
Admin dapat masuk ke sistem menggunakan username dan password untuk mengakses fitur manajemen konveksi.

2. Manajemen Data Pelanggan

Admin dapat:
- Menambah data pelanggan
- Mengedit data pelanggan
- Menghapus data pelanggan
- Melihat daftar pelanggan

3. Manajemen Pesanan

Admin dapat mencatat pesanan pelanggan dengan informasi:
- Nama pelanggan
- Produk yang dipesan
- Jumlah pesanan
- Ukuran produk
- Status pesanan

4. Proses Produksi
- Admin dapat mengelola proses produksi pesanan seperti:
- Pesanan diterima
- Dalam proses produksi
- Pesanan selesai

5. Manajemen Pembayaran
- Admin dapat mencatat pembayaran pelanggan baik berupa
- Uang muka (DP)
- Pelunasan

6. Laporan Sistem
Owner dapat melihat laporan:
- Laporan penjualan
- Laporan produksi
- Teknologi yang Digunakan

Bahasa Pemrograman: PHP

Database: MySQL

Frontend: HTML, CSS, Bootstrap, PHP, JS

Web Server: Apache (Laragon)

Aktor dalam sistem:

Admin

- Mengelola data pelanggan
- Mengelola data produk
- Mengelola pesanan
- Mengelola proses produksi
- Mengelola pembayaran

- Melakukan pemesanan produk
- Melihat status pesanan
